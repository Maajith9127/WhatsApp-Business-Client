# Express App
Express.js application for WABAS

## Installation

```bash
git clone https://github.com/yourusername/your-express-app.git
cd your-express-app
npm install
````

## Running the App

* For development:

```bash
npm run dev
```

* For production:

```bash
npm start
```

## Setup Environment Variables

Create a `.env` file in the root directory and add the following variables:

```env
DATABASE_NAME=
DATABASE_HOST=
DATABASE_USER=
DATABASE_PASSWORD=
WABA_ACCESS_TOKEN=
WABA_BUSINESS_ACCOUNT_ID=
RESEND_API_KEY=
FRONTEND_URL=
```
